    Welcome to the COVID Border Accountability Project

    This map tool displays hand-coded information on border closures around the world introduced in response to the COVID-19 pandemic. Clicking on a country will provide more info on policies, including sources.

    Only 2020 data has been verified by our team. For the most up-to-date travel rules, we recommend checking with the government website of the host country. For research, we recommend further verifying the 2021 data prior to publication.
